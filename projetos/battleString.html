Battle String

Sobre o Projeto

O Battle String é um jogo criado para aprimorar habilidades de digitação de forma divertida. Através de desafios gamificados e visual retrô, ele ajuda usuários a melhorar velocidade e precisão, gerando relatórios de desempenho. Além do uso individual, o jogo também pode ser adotado por empresas e instituições de ensino como ferramenta de treinamento.

Tecnologias Utilizadas:

  • Bootstrap
  • ORM
  • Canvas Element
  • JavaScript
  • Html
  • Css
  • Random Word API
