Find Bus 1.0

Sobre o Projeto

O FindBus é um projeto criado para quem depende do transporte público em São Paulo e enfrenta falta de informações em tempo real e dificuldade para escolher o melhor trajeto. A solução oferece uma forma prática de planejar viagens, permitindo visualizar linhas de ônibus, descobrir novos caminhos e salvar favoritos. Com isso, o FindBus torna o deslocamento mais rápido, seguro e eficiente.

Tecnologias Utilizadas:

  • React Native
  • Node.js
  • Postgree
  • API SpTrans
